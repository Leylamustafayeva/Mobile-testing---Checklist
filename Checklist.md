**Checklist**

**Functional Testing**
- [ ] Forgot password funksiyası uğurla test edildi.  
- [ ] Logout işləyir.  
- [ ] Menu düymələri işləyir və müvafiq səhifəyə yönləndirir.  
- [ ] Payment Testing - İstifadəçi asanlıqla lazımi ödənişi (internet və s.) edə bilir və nəticədə istifadəçi balansı azalır.  
**UI Testing**
- [ ] Dark mode və light mode yoxlanışı.  
- [ ] Rotate etdikdə responsivliyi düzgün işləyir.  
- [ ] Scroll işləyir.  
- [ ] Buttonlar işləyir.  
**Performance Testing**
- [ ] App-ın açılma sürəti normaldır.  
- [ ] Səhifə yüklənmə sürəti normaldır.  
- [ ] Uzun müddət istifadə zamanı stabilliyini saxlayır.  
**Security Testing**
- [ ] Biometric Authentication uğurla test edildi.  
- [ ] Bir neçə səhv login cəhdindən sonra bloklanma.  
- [ ] Yanlış şifrə / istifadəçi adı error verir.  
- [ ] Şifrə gizli göstərilir (***).  
- [ ] Boş sahə buraxıldıqda müvafiq xəta bildirişi verir.  
**Usability Testing**
- [ ] İstifadəçi rahatlığı təmin olunub.  
- [ ] Addımlar sadədir.  
- [ ] Mətnlərin font ölçüləri və font növü oxunaqlıdır.  
- [ ] Düymələr aydın şəkildə işarələnib.  
- [ ] Rəng kontrastı uyğun seçilib.  
- [ ] Şəkillər və ikonlar üçün alternativ mətnlər qeyd olunub.  
**Compatibility Testing**
- [ ] Tətbiq müxtəlif mobil cihazlarda yoxlanıldı.  
- [ ] Müxtəlif ekran ölçülərində yoxlanıldı.  
- [ ] Müxtəlif brauzerlərdən uğurla keçdi (Safari, Chrome, Firefox).  
**Notification Testing**
- [ ] Notification klik etdikdə düzgün səhifəyə yönləndirir.  
- [ ] Birdən çox notification düzgün göstərir.  
- [ ] Notification, tətbiq backgroundda və ya açıq olduğu zaman gəlir.  
- [ ] Notification gələn zaman uyğun tətbiqin ikonu əks olunur.  
**Network Testing**
- [ ] İnternet olmayanda tətbiq davranışı yoxlanıldı.  
- [ ] Zəif internetdə tətbiqin davranışı yoxlanıldı.  
- [ ] İnternet kəsildikdə error mesajı verilir.  
- [ ] Yenidən internet qoşulduqda tətbiq bərpa olunur.  
- [ ] Mobile data və Wi-fi keçidi yoxlanıldı.  
- [ ] Airplane mode-da tətbiqin davranışının yoxlanılması.  
- [ ] Push notification-ların internet bağlantısından asılılığının yoxlanması.
